# 📚 Udemy Clone (HTML & CSS)

This project is a front-end clone of the popular online learning platform **Udemy**, built using pure **HTML** and **CSS**. It replicates the visual layout and user interface of Udemy, focusing on responsive design, clean structure, and modern styling techniques.

## 🚀 Features
- 🎯 Responsive homepage layout  
- 📖 Course listing sections with cards  
- 🔍 Navigation bar with search functionality (UI only)  
- 🧭 Category browsing interface  
- 💻 Clean and structured HTML5 markup  
- 🎨 Styled using CSS3 (Flexbox & Grid)

## 🛠️ Technologies Used
- HTML5  
- CSS3  

## 📌 Purpose
This project was created to practice front-end development skills, especially layout design, responsiveness, and UI replication from real-world platforms.

## ⚠️ Disclaimer
This is a **static clone** created for educational purposes only. It does not include backend functionality or real course data and is not affiliated with Udemy.
